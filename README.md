## 📘 프로젝트 소개
시각장애인을 위한 점자 기반 채팅 인터페이스입니다.

## 🛠 주요 기능
- 점자 인코딩/디코딩
- 점자 디바이스 연동
- 아두이노 기반의 IOT 프로젝트

## 🧾 설치 방법

1. Git을 다운로드 합니다.
   처음이신분은 아래 링크에서 다운로드 하는 방법을 참고하세요!
